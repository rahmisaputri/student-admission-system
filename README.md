# Sistem Pendaftaran Mahasiswa Berbasis Web
Sistem Pendaftaran Mahasiswa Berbasis Web merupakan aplikasi yang dikembangkan untuk memudahkan proses pendaftaran calon mahasiswa secara online. Sistem menyediakan dua jenis pengguna, yaitu Admin dan Pendaftar, sehingga proses pengelolaan data menjadi lebih efisien.

# Fitur
**Pendaftar**
- Registrasi akun
- Login
- Mengisi formulir pendaftaran yaitu biodata, asal sekolah, dan nilai
- Mengunggah dokumen 
- Mendapat nomor pendaftaran
- Logout

**Admin**
- Login
- Melihat seluruh data pendaftar
- Menambah data
- Mengubah data
- Menghapus data
- Logout

# Teknologi yang Digunakan
**Backend**
- PHP
  
**Frontend**
- CSS
- HTML
  
**Database**
- MySQL
  
**Web Server**
- XAMPP
